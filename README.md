# Chlorhexidine Release Calculator

Программа для обработки данных по высвобождению хлоргексидина из модифицированных покрытий/поверхностей по данным спектрофотометрии.

Основная задача программы - автоматизировать расчёт концентрации вещества по оптической плотности.

## Что делает программа

Программа позволяет:

* вводить экспериментальные данные в формате «время — оптическая плотность»;
* рассчитывать концентрацию хлоргексидина по калибровочной зависимости;
* учитывать отбор проб из раствора;
* рассчитывать суммарное высвобождение;
* рассчитывать процент высвобождения относительно начального количества вещества;
* строить график зависимости высвобождения от времени;
* сохранять результаты в таблицу Excel.



## Используемые данные

На вход подаются две колонки:

| Время, мин | Оптическая плотность |
| ---------: | -------------------: |
|          1 |             0.119310 |
|          5 |             0.112820 |
|         10 |             0.119000 |
|         15 |             0.196030 |

Данные можно вставлять напрямую в ячейку Google Colab без загрузки Excel-файла.

## Основная формула

Концентрация рассчитывается по калибровочной зависимости:

```text
A = k · C + b
```

где:

```text
A — оптическая плотность;
C — концентрация вещества;
k — коэффициент наклона калибровочного графика;
b — свободный член.
```

Тогда:

```text
C = (A - b) / k
```

Значения `k` и `b` задаются в начале программы и могут быть изменены под конкретную калибровку.

## Учёт отбора проб

Если в ходе эксперимента из раствора отбирались пробы, программа может учитывать потерю вещества при каждом отборе.

Это важно, потому что после каждой пробы часть раствора удаляется, а вместе с ним удаляется и часть высвободившегося хлоргексидина. Без учёта этого фактора суммарное высвобождение может быть рассчитано некорректно. :(




## Запуск в Google Colab

1. Открыть ноутбук `.ipynb` в Google Colab.
2. Вставить экспериментальные данные в ячейку с таблицей.
3. Проверить параметры:

   * коэффициенты калибровки;
   * объём раствора;
   * объём отбираемой пробы;
   * начальное количество хлоргексидина.
4. Запустить все ячейки.
5. Получить таблицу, график и Excel-файл с результатами.
